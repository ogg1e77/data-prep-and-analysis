# Лабораторна робота №2 — Підготовчий етап аналізу даних

## Опис завдання

Робота складається з двох частин:

**Частина 1** — завантаження та обробка даних VHI-індексу (Vegetation Health Index) по регіонах України з відкритої бази NOAA:
- завантаження 27 файлів через `urllib` з антиколізійним механізмом
- data cleaning: видалення зайвого тексту, заповнення пропусків медіаною
- переіндексація областей з англійської абетки (NOAA) на українську
- формування вибірок: по року, по діапазону років, статистика (min/max/mean/median), роки з екстремальною посухою

**Частина 2** — аналіз датасету Individual Household Electric Power Consumption (UCI):
- автоматичне завантаження датасету через `urllib`
- data cleaning: приведення типів, об'єднання дати і часу, заповнення пропусків
- формування 4 вибірок за різними умовами
- нормалізація (Min-Max) та стандартизація (Z-score)
- коефіцієнти кореляції Пірсона та Спірмена
- One Hot Encoding категоріального атрибута

## Встановлення та запуск

### 1. Клонувати репозиторій
```bash
git clone https://github.com/<username>/<repo>.git
cd <repo>/lab-2
```

### 2. Створити віртуальне середовище
```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS / Linux
source venv/bin/activate
```

### 3. Встановити залежності
```bash
pip install -r requirements.txt
```

### 4. Запустити Jupyter
```bash
jupyter notebook
```

## Отримання датасетів

| Датасет | Як отримується |
|:---|:---|
| VHI (NOAA) | **Автоматично** при запуску `part1_vhi.ipynb` — скрипт завантажує 27 файлів у папку `vhi_data/` |
