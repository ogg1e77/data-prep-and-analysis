# Лабораторна робота №2 

## Опис

Робота складається з двох частин:

- **Частина 1** — завантаження та аналіз VHI-індексу (Vegetation Health Index) по регіонах України з відкритої бази NOAA.
- **Частина 2** — аналіз датасету споживання електроенергії домогосподарством (UCI Repository).

---

## Структура

```
lab2/
├── part1_vhi.ipynb              # Частина 1: VHI-дані
├── part2_power.ipynb            # Частина 2: споживання електроенергії
├── vhi_data/                    # Створюється автоматично при запуску part1
└── household_power_consumption.txt  # Завантажується вручну (див. нижче)
```


---

## Швидкий старт

### 1. Клонувати репозиторій

```bash
git clone https://github.com/<username>/<repo>.git
cd <repo>/lab2
```

### 2. Створити та активувати віртуальне середовище

```bash
python -m venv venv

# Windows
venv\Scripts\activate

# macOS / Linux
source venv/bin/activate
```

### 3. Встановити залежності

```bash
pip install jupyter pandas numpy scipy
```

### 4. Завантажити датасет для Частини 2

Перейдіть на сторінку датасету та завантажте архів:

**[Individual Household Electric Power Consumption — UCI Repository](https://archive.ics.uci.edu/dataset/235/individual+household+electric+power+consumption)**

Розпакуйте та покладіть файл `household_power_consumption.txt` у папку `lab2/`.

### 5. Запустити Jupyter

```bash
jupyter notebook
```

Відкрийте `part1_vhi.ipynb` або `part2_power.ipynb` та запускайте комірки послідовно.

> VHI-дані для Частини 1 завантажуються **автоматично** при першому запуску — вручну нічого робити не потрібно.

## Зміст Частини 1 (`part1_vhi.ipynb`)

| № | Завдання |
|---|---------|
| 1 | Завантаження VHI-файлів для 27 областей через `urllib` з антиколізійним механізмом |
| 2 | Data cleaning: видалення зайвого тексту, заповнення пропусків, нормалізація колонок |
| 3 | Переіндексація областей: NOAA (англ. алфавіт) → українська абетка |
| 4.1 | Вибірка: ряд VHI для конкретної області за рік |
| 4.2 | Вибірка: VHI за діапазон років для кількох областей |
| 4.3 | Вибірка: min, max, mean, median для заданих областей і років |
| 4.4 | Додаткова: роки з екстремальною посухою (VHI < 15) |

## Зміст Частини 2 (`part2_power.ipynb`)

| № | Завдання |
|---|---------|
| 1 | Завантаження та відкриття датасету |
| 2 | Data cleaning з профілюванням часу (`timeit`) |
| 3.1 | Записи з активною потужністю > 5 кВт |
| 3.2 | Записи з силою струму 19–20 А, де пральна + холодильник > бойлер + кондиціонер |
| 3.3 | Випадкова вибірка 500 000 записів, середні по 3 групах споживання |
| 3.4 | Записи після 18:00 з > 6 кВт, де група 2 найбільша; кожен 3-й з 1-ї пол. + кожен 4-й з 2-ї |
| 4 | Min-Max нормалізація та Z-score стандартизація |
| 5 | Коефіцієнти кореляції Пірсона та Спірмена |
| 6 | One Hot Encoding категоріального атрибута `hour_period` |
