# Лабораторна робота №6 — C++ OpenCV

## Опис завдання

Програма читає відеопотік з камери в реальному часі, відображає його у вікні та реагує на натискання клавіатури й миші, перемикаючи різні режими обробки зображення.

## Архітектура

| Клас | Файли | Призначення |
|------|-------|-------------|
| `CameraProvider` | `CameraProvider.hpp/.cpp` | Обгортка над `cv::VideoCapture`, метод `getFrame()` |
| `KeyProcessor` | `KeyProcessor.hpp/.cpp` | Обробка клавіш, enum `Mode`, керування зумом/обертанням/зсувом |
| `FrameProcessor` | `FrameProcessor.hpp/.cpp` | Обробка кадру відповідно до режиму, обробка миші, слайдер яскравості |
| `Display` | `Display.hpp/.cpp` | Відображення (`cv::imshow`), trackbar яскравості |
| `main` | `main.cpp` | Створення об'єктів, головний цикл `while` |

## Функціонал (режими — клавіші 0–7)

| Клавіша | Режим | Опис |
|---------|-------|------|
| `0` | NORMAL | Без обробки |
| `1` | INVERT | Інверсія кольорів (`bitwise_not`) |
| `2` | BLUR | Gaussian Blur |
| `3` | CANNY | Canny-фільтр (виявлення контурів) |
| `4` | SOBEL | Фільтр Собеля |
| `5` | BINARIZE | Бінаризація (порогова обробка) |
| `6` | GLITCH | Розсування каналів B/R у різні боки |
| `7` | PIP | Picture-in-picture (PNG-зображення на відео) |

## Керування

| Клавіша / дія | Ефект |
|---------------|-------|
| Стрілочки `←↑→↓` | Зсув зображення (у режимі PIP — рух картинки-в-картинці) |
| `+` / `-` | Зум (наближення/віддалення) |
| `,` / `.` | Плавне обертання |
| `r` | Скинути всі трансформації (зум/зсув/обертання) |
| `q` / `ESC` | Вихід з програми |
| ЛКМ (натиснути → вести → відтиснути) | Малює прямокутник або лінію |
| ПКМ | Перемикає тип фігури: прямокутник ↔ лінія |
| Колесо миші | Додатковий зум (комбінується з `+`/`-`) |
| Слайдер «Brightness» у вікні | Регулює яскравість зображення в реальному часі |

На екрані постійно відображаються: поточний режим, FPS, середня інтенсивність каналів B/G/R та підказка з клавішами.

## Вимоги до системи

- **ОС:** Ubuntu 20.04+ / інший Linux з apt (для `preinstall.sh`)
- **Компілятор:** GCC 9+ (підтримка C++17)
- **CMake:** 3.20+
- **OpenCV:** 4.x (встановлюється через `preinstall.sh`)
- **Залізо:** будь-яка веб-камера, підтримувана `cv::VideoCapture` (USB або вбудована)

## Встановлення, білд та запуск

```bash
git clone <repo_url>
cd <repo>/lab-6

chmod +x preinstall.sh build.sh run.sh

./preinstall.sh   # встановлення OpenCV, CMake, build-essential
./build.sh        # CMake + make → build/lab6_opencv
./run.sh          # запуск програми
```

## Примітка щодо picture-in-picture (режим 7)

Покладіть будь-яке `.png`-зображення у `assets/overlay.png` — воно буде показане як picture-in-picture у режимі `7`. Якщо файл відсутній, режим `7` просто не накладає додаткове зображення (решта функціоналу працює без змін).
