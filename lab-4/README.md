# Лабораторна робота №4 — Візуалізація даних 2

## Опис завдання

Інтерактивна програма для побудови графіка гармоніки виду:

```
y(t) = A * sin(ω*t + φ)
```

з накладеним шумом та фільтрацією за допомогою фільтра Баттерворта (low-pass).

## Функціонал

| Елемент | Опис |
|---------|------|
| Слайдер Amplitude | Амплітуда гармоніки (0.1 – 5.0) |
| Слайдер Frequency | Частота гармоніки (0.1 – 5.0 Гц) |
| Слайдер Phase | Фазовий зсув (0 – 2π) |
| Слайдер Noise Mean | Середнє значення шуму (-1.0 – 1.0) |
| Слайдер Noise Covariance | Дисперсія шуму (0.01 – 2.0) |
| Слайдер Cutoff Frequency | Частота зрізу фільтра (0.1 – 10.0 Гц) |
| Чекбокс Show Noise | Вмикає/вимикає відображення шуму |
| Чекбокс Show Filtered | Вмикає/вимикає відфільтровану гармоніку |
| Кнопка Reset | Відновлює початкові параметри |

**Важливо:**
- Зміна параметрів гармоніки не перегенеровує шум
- Зміна параметрів шуму перегенеровує лише шум

## Вимоги до системи

- **Python:** 3.10+
- **ОС:** Windows / macOS / Linux

## Встановлення та запуск

```bash
# 1. Створити віртуальне середовище

venv\Scripts\activate
# macOS / Linux
source venv/bin/activate

# 2. Встановити залежності
pip install -r requirements.txt

# 3. Запустити програму
python harmonic.py
```
