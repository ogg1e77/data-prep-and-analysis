# Лабораторна робота №5 — Наука про дані: обмін результатами та початковий аналіз

## Опис завдання

Веб-додаток на **Streamlit** для інтерактивного аналізу VCI / TCI / VHI-даних по регіонах України (датасет з лабораторної роботи №2).

## Функціонал

| Елемент | Опис |
|---------|------|
| Dropdown «Часовий ряд» | Вибір VCI, TCI або VHI |
| Dropdown «Область» | Вибір однієї з 27 областей України |
| Slider «Інтервал тижнів» | Фільтрація за тижнями (1–52) |
| Slider «Інтервал років» | Фільтрація за роками (1981–2024) |
| Checkbox «Сортувати за зростанням» | Сортування таблиці за обраним індексом |
| Checkbox «Сортувати за спаданням» | Сортування таблиці за обраним індексом (у зворотному порядку) |
| Button «Reset» | Повертає всі фільтри до початкових значень |

**Вкладки:**
1. **Таблиця** — відфільтровані дані за обраними параметрами
2. **Графік** — часовий ряд обраного індексу для обраної області
3. **Порівняння областей** — середнє значення індексу по всіх областях за вказаний період, обрана область підсвічена

**Логіка сортування:** якщо увімкнені одразу два чекбокси — застосовується сортування за зростанням (про це виводиться попередження користувачу).

**Розташування:** інтерактивні елементи — у лівій колонці, таблиця та графіки — у правій.

## Вимоги до системи

- **Python:** 3.10+
- **ОС:** Windows / macOS / Linux

## Встановлення та запуск

```bash
# 1. Створити віртуальне середовище
python -m venv venv

# Windows
venv\Scripts\activate
# macOS / Linux
source venv/bin/activate

# 2. Встановити залежності
pip install -r requirements.txt

# 3. Запустити застосунок
streamlit run app.py
```

Додаток відкриється автоматично в браузері за адресою `http://localhost:8501`.

## Отримання даних

Дані VHI/VCI/TCI завантажуються **автоматично** з NOAA при першому запуску застосунку (та ж логіка, що в лабораторній роботі №2). Якщо джерело NOAA тимчасово недоступне, застосунок використовує згенеровані тестові дані з тією ж структурою, щоб демонстрація функціоналу залишалась можливою.
